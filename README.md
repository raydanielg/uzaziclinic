# AfyaCare Pro - Health Management System

Mfumo wa kisasa wa usimamizi wa hospitali na duka la dawa.

## 🔑 Demo Credentials (Roles & Logins)

Unaweza kutumia akaunti hizi za majaribio kuingia kwenye mfumo na kuona panels tofauti. 
**Nenosiri (Password) kwa akaunti zote ni:** `password`

| Role | Email | Dashboard Features |
|------|-------|-------------------|
| **Admin** | `admin@afyacare.com` | Full Control, User Mgmt, System Settings |
| **Doctor** | `doctor@afyacare.com` | Appointments, Patients, Prescriptions, Lab Requests |
| **Nurse** | `nurse@afyacare.com` | Vitals Recording, Patient Check-in, Ward Mgmt |
| **Pharmacist** | `pharmacist@afyacare.com` | Inventory, Stock Mgmt, Process Prescriptions |
| **Lab Tech** | `lab@afyacare.com` | Lab Tests, Results Entry, Equipment Tracking |
| **Accountant** | `accountant@afyacare.com` | Billing, Invoices, Receipts, Insurance Claims |
| **Receptionist** | `receptionist@afyacare.com` | Patient Registration, Booking, Queue Mgmt |
| **Patient** | `patient@afyacare.com` | Booking, History, Pharmacy Store, My Orders |

## 🚀 Sifa za Mfumo

- **Role-Based Access Control (RBAC):** Kila mtumiaji anaona menu na taarifa zake tu.
- **Smart Sidebar:** Sidebar inabadilika kulingana na role ya aliye-login.
- **Modern UI:** Imejengwa kwa Bootstrap 5, FontAwesome 6, na Animate.css.
- **Security:** Inatumia Laravel Sanctum na Middleware kwa ulinzi wa data.

## 🛠️ Jinsi ya Ku-setup
1. `composer install`
2. `npm install`
3. `php artisan migrate --seed` (Hii itaunda tables na kuweka hizi akaunti za demo)
4. `php artisan serve`
